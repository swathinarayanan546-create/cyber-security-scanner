# 🛡️ Cyber Security Scanner

This is a Python-based mini project that analyzes website security.

## 🔍 Features
- Checks HTTP/HTTPS protocol
- Identifies missing security headers
- Scans common open ports
- Generates a report file

## 💻 Technologies Used
- Python
- requests module
- socket module

## 🚀 How to Run
python scanner.py
